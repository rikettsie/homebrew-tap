# rikettsie/tap

Homebrew tap for tools by [rikettsie](https://github.com/rikettsie).

## Install

    brew tap rikettsie/tap

## Available formulae

| Formula | Description |
|---------|-------------|
| `rdrop` | P2P streamed file transfer with ring-based access control. Built on iroh. |

## Usage

    brew install <formula>
    brew upgrade <formula>
